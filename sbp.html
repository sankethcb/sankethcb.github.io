Soft-Body Physics Simulation

Embedded Game

Description

A demonstration of the math behind soft-body physics implemnting Hooke's Law. You can create 2D polygons linked by springs that can be dragged around to exhibit interactions with other objects, gravity and the ground.

Details

  • Made entirely in Unity3D.
  • Spacebar spawns a polygon on the mouse.
  • Drag objects using the red points.
  • C toggles collisions on/off.
  • Number keys 3-6 change the number of sides of the spawned polygon.
