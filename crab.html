Crab Rhapsody [In Development]

Description

Crab Rhapsody is a fast-paced 3D action game where a lone crab warrior must ascend a deadly tower to rescue the kidnapped memebers of his favorite rock band!

Combines interactive music based on the band members you have rescued and a high-flying physics sand box mechanic to hook in players, along with the expressive titular player character.

Details

  • Made for a graduate production class that followed the agile development process, with distinct sprints and task breakdowns for each of the major milestones: Vertical Slice, Alpha, Beta & Release Candidate.
  • Co-designer & engineer, with a team size of 3 including our Composer.
  • Responsible for all systems & technical design, and in-engine implementation including the following systems/components:
    • Kinematic Character Controller with custom collision detection.
    • Core physics sandbox mechanic with custom force resolution.
    • Grab/throw system, along with weapon interaction.
    • Procedural level generation.
    • Enemy AI behavior.
  • Integrated and currently utilizing Wwise as the audio middleware.
  • Optimizing the game through both code and asset management, keeping track of framrate, draw calls and screen tricounts.
  • Developed for PC (Windows)
